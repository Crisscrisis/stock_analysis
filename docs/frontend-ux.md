## 二、当前交互行为（实现状态）

### 2.1 自选股侧边栏

| 行为 | 当前实现 |
|------|---------|
| 默认股票 | 纳斯达克 100 前 20 只（硬编码） |
| 点击切换 | 点击任意行，右侧图表切换到该股票 |
| 选中状态 | 左侧 2px 蓝色竖条 + `bg-bg-hover` 背景 |
| 删除 | hover 时右侧出现 ✕ 按钮，点击从自选股移除 |
| 实时价格 | 当前查看的股票通过 WebSocket 更新价格；侧边栏其他股票价格无实时更新 |
| 添加股票 | 通过顶部搜索框搜索后选择，自动加入侧边栏 |

### 2.2 顶部搜索栏

| 行为 | 当前实现 |
|------|---------|
| 触发搜索 | 输入后 300ms 防抖，调用后端搜索接口 |
| 搜索范围 | 仅 A 股（akshare），不支持直接搜索美股 |
| 选择结果 | 切换图表到该股票，同时自动加入自选股 |
| 关闭下拉 | 选择后关闭；点击外部区域**不会**关闭（当前缺陷） |

### 2.3 K 线图（ChartPanel）

| 行为 | 当前实现 |
|------|---------|
| 切换股票 | 重新请求数据，图表重绘 |
| 周期切换 | 1M / 3M / 6M / 1Y / 3Y，切换后重新请求 |
| 指标切换 | MA / MACD / RSI / BOLLINGER，多选，切换后重新请求 |
| 指标显示 | 叠加在同一坐标轴（MACD/RSI 使用独立 priceScaleId） |
| 加载状态 | 显示"加载中…"文字 |
| 错误状态 | 显示"加载数据失败"文字 |

### 2.4 底部面板

| 面板 | 行为 |
|------|------|
| 基本面 | 随选中股票切换，显示 PE/PB/市值/营收/净利润/股息率 |
| 资金流向 | 随选中股票切换；美股显示"仅支持 A 股"提示 |

## 四、已确认改进方案（待实现）

### 4.1 侧边栏选中状态 ✅ 已确认

**问题**：当前 2px 蓝色左边框 + hover 背景，深色主题下辨识度低。

**采用方案**：
- 选中行背景色改为 `bg-accent-blue/20`（蓝色透明叠加）
- 左侧竖条加宽至 **4px**，颜色改为更亮的 `#60a5fa`
- 股票代码文字选中时变为 **纯白加粗**
- 股票名称选中时颜色变为 `text-slate-300`（原为 `text-accent-gray`）

**改动文件**：`src/components/WatchlistSidebar.tsx`

---

### 4.2 删除按钮 ✅ 已确认

**问题**：✕ 只在 hover 时出现，用户不知道可以删除。

**采用方案**：
- ✕ 按钮**始终可见**
- 非 hover 状态：`text-slate-600`（深灰，低调但存在）
- hover 状态：`text-red-400`（红色，明确表示删除）
- 按钮尺寸从 `text-xs` 改为 `text-sm`，更易点击

**改动文件**：`src/components/WatchlistSidebar.tsx`

---

### 4.3 图表区当前股票标题 ✅ 已确认

**问题**：用户不清楚右侧图表对应哪只股票。

**采用方案**：
- 在 ChartPanel 工具栏左侧增加**当前股票标识**：显示 `代码 · 股票名称`
- 字号 `text-sm`，颜色白色加粗，与周期/指标按钮同行
- 名称从 watchlist 数据中取，没有名称时只显示代码

**改动文件**：`src/components/ChartPanel.tsx`，`src/App.tsx`（传入 name prop）

---

### 4.4 搜索框点击外部关闭 ✅ 已确认

**问题**：下拉结果打开后，点击页面其他区域不会关闭。

**采用方案**：
- 使用 `useRef` 监听 `document` 的 `mousedown` 事件
- 判断点击目标是否在搜索框容器外，若是则关闭下拉

**改动文件**：`src/components/SearchBar.tsx`

## 五、待讨论 / 待决策

### 5.1 MACD / RSI 显示位置
> **问题**：叠加在主图导致 Y 轴刻度混乱，可读性差。

候选方案：
- **方案 A**（当前）：全部叠加在同一图表，使用独立 priceScaleId
- **方案 B**：MACD / RSI 显示在主图**下方**独立子图，固定高度比例（主图 70%，子图 30%）
- **方案 C**：用 Tab 切换，一次只看一个指标

**状态**：待讨论 ✏️

---

### 5.2 侧边栏其他股票实时价格
> **问题**：目前只有当前查看的股票有 WebSocket 实时价格，其余显示"—"。

候选方案：
- **方案 A**：只保持当前方式（仅当前股票实时），其余价格通过 REST 轮询（每 30s）
- **方案 B**：为自选股列表中所有股票建立 WebSocket 连接（并发连接数多）
- **方案 C**：切换股票时才拉取该股票的最新报价，不做实时推送

**状态**：待讨论 ✏️

---

### 5.3 美股直接输入代码添加
> **问题**：搜索框目前只对接 A 股搜索，用户无法通过搜索找到 TSLA 等美股。

候选方案：
- **方案 A**：搜索无结果时，若输入内容符合美股代码格式（纯字母），提示"直接添加 TSLA 到自选股"
- **方案 B**：搜索框支持多数据源，后端同时搜索 A 股 + 美股
- **方案 C**：在侧边栏底部增加"手动添加"入口，允许直接输入代码和市场

**状态**：待讨论 ✏️
